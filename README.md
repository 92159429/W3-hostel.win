# W3 Hostel GitHub Pages Repo

這是可直接上傳到 GitHub 的靜態網站 repo。

## 結構

- `index.html` 首頁
- `w3.html` W3 介紹
- `rooms-services.html` 客房及服務
- `travel-focus.html` 旅遊聚焦
- `contact-access.html` 聯絡與交通
- `assets/css/styles.css` 網站樣式
- `assets/js/site.js` 多語系與頁面動畫
- `assets/images/` 網站圖片
- `CNAME` 自訂網域設定
- `.nojekyll` 關閉 Jekyll 處理
- `robots.txt` 搜尋引擎規則
- `sitemap.xml` 網站地圖

## GitHub 上傳方式

1. 在 GitHub 建立一個新 repository，例如 `w3-hostel`。
2. 將此資料夾全部內容上傳到 repository 根目錄。
3. 進入 GitHub repository 的 `Settings` > `Pages`。
4. Source 選擇 `Deploy from a branch`。
5. Branch 選擇 `main`，資料夾選擇 `/ (root)`。
6. 儲存後等待 GitHub Pages 部署完成。
7. 若使用自訂網域 `w3-hostel.win`，請確認 DNS 已正確指向 GitHub Pages，且保留本 repo 中的 `CNAME` 檔案。
