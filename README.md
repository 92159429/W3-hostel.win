## 結構

- `index.html` 首頁
- `w3.html` W3
- `space.html` 空間
- `travel.html` 旅遊資訊
- `contact.html.html` 聯絡
- `assets/css/styles.css` 網站樣式
- `assets/js/site.js` 多語系與頁面動畫
- `assets/images/` 網站圖片
- `CNAME` 自訂網域設定
- `.nojekyll` 關閉 Jekyll 處理
- `robots.txt` 搜尋引擎規則
- `sitemap.xml` 網站地圖s

## 已設定內容

- 多頁版網站
- 多語系：正體中文、English、Français、Español、Deutsch、日本語、한국어
- GitHub Pages 相容
- Google Analytics
